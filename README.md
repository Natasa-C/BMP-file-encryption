# File-Encryption

Description

## Compile and run
### encryption_script.c
> To compile:  	gcc encryption_script.c encryption_functions.c -o script

> To run:		    ./script

### encryption_verifier.c
> To compile:  	gcc encryption_verifier.c -o verifier

> To run:		    ./verifier

> Name of encrypted file to test: output/BMPencrypted.bmp
